Trung tâm công cụ Bongstudio

Quy trình chuẩn: Bước 1 tạo JSON + ảnh từ link sản phẩm → Bước 2 gửi JSON + ảnh sang Worker để đẩy lên repo.
Bước 1
Tạo JSON + HTML + ảnh (Product Builder)
Dán link sản phẩm từ Shopee/Lazada/TikTok/Tiki, nhập tên + giá → tool sinh JSON, HTML card, poster share và WebP thumb để up repo.
Mở Bongstudio Product Builder
Gợi ý: làm xong 1 sản phẩm ở đây rồi chuyển sang bước 2 để đẩy lên repo tự động.
Bước 2
Gửi JSON + ảnh sang Worker (Worker Bridge)
Dán JSON từ Bước 1 + chọn WebP thumb đã tải về → Worker nhận JSON/ảnh và commit thẳng vào repo bongstudio68.github.io.
Mở Bongstudio Worker Bridge
Cần cấu hình trước: URL Worker, repo, thư mục assets/json/bongstudio và assets/img/bongstudio.
Đây là trang nội bộ dành cho MXD/Bongstudio — không index trên Google, chỉ dùng để thao tác tool nhanh trên PC/điện thoại.
